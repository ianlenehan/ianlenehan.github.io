How To Look Like a Real Developer

Put Down That Mouse

Browser Shortcuts

  • Select a tab: CMD - 3 (tab number)
  • Cycle through tabs in Chrome: CMD - option - left/right
  • Cycle through tabs in Safari: CMD - option - shift - left/right
  • Warn before quitting (Chrome) - Chrome > Warn Before Quitting
  • Select address bar: CMD - L

OS Keyboard Shortcuts

Navigating the cursor

  • Move cursor to start of a line: ctrl - A or CMD - left
  • Move cursor to end of a line: ctrl - E or CMD - right
  • Bypass words with the cursor: option - left/right
  • Delete whole words: option - delete

OS Keyboard Shortcuts

Editing text

  • Select text from cursor to start of a line: CMD - shift - left
  • Select text from cursor to end of a line: CMD - shift - right
  • Select individual words with the cursor: option - shift - left/right
  • Move entire lines of text around: CMD - ctrl - up/down
