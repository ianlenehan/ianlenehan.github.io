OS Keyboard Shortcuts

Navigating the cursor

Move cursor to start of a line

ctrl - A or CMD - left

Move cursor to end of a line

ctrl - E or CMD - right

Navigating the cursor

Bypass words with the cursor

option - left/right

Delete a word up to the cursor

option - delete

Editing text

Select text from cursor to start of a line

CMD - shift - left

Select text from cursor to end of a line

CMD - shift - right

Editing text

Select individual words with the cursor

option - shift - left/right

Move entire lines of text around

CMD - ctrl - up/down

GIT

Aliases

add to .gitconfig

  • g = git
  • co = checkout
  • ci = commit
  • st = status
  • br = branch

You can also add other terminal aliases to .bash_profile

e.g. shortcuts to certain directories

Checkout the previous branch with -

git checkout - or g co -
